# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [5.0.0] - 2024-01-01

### Added
- Full Unsplash API wrapper for .NET 8.0
- OAuth 2.0 and Bearer Token authentication
- Built-in Polly retry policies for transient fault handling
- Configurable timeout, retries, and retry delay
- XML documentation for all public API members
- Comprehensive test suite with 57 tests

### Dependencies
- Polly 8.5.2
- Microsoft.Extensions.Http.Polly 9.0.0
